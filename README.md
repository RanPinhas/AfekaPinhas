# Project Description
This project was built as part of the first homework assignment in the course. The website showcases 5 significant places in Israel, using only the basic HTML and CSS technologies taught in class. The layout of the content pages is based on HTML tables, and the homepage features navigation using an Image Map.

# The Choice of Places: Connection to the Sea and the "Mother of the Moshavot"
The central theme guiding my choice of locations is my deep connection to the sea. As someone who served in the Israeli Navy and as a licensed skipper who loves the maritime world, the sea is an inseparable part of my life. Therefore, most of the sites I chose to present are the main marinas and ports of Israel (Haifa Port, Herzliya Marina, Ashkelon Marina, and Eilat Marina).

Alongside my love for the sea, I also chose to dedicate a place of honor to Petah Tikva, known as the "Mother of the Moshavot" (Em HaMoshavot). This city represents the historical and Zionist roots of the country, and it was important to me to include it as a central landmark in the Israeli landscape.
